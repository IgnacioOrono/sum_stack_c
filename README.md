# Stack Calculator en C
Proyecto de una calculadora que utiliza una estructura de pila (Stack) para realizar operaciones. Se uso como ejercicio para practicar el manejo de estructuras y punteros en lenguaje C.

## Funcionalidades
* **Push:** Agregar un número entero a la pila.
* **Pop:** Quitar el último número ingresado.
* **Sumar:** Sumar los últimos dos números de la pila.
* **Stack Dump:** Visualización del estado actual de la memoria y la posición del puntero de pila (SP).

## Compilación y ejecución

```bash
make && ./calculadora
```

## Estructura del código
* `stack.h`: Definición de la estructura y funciones.
* `stack.c`: Lógica de manejo de la pila y operaciones.
* `main.c`: Menú interactivo.
